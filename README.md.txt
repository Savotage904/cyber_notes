# Cyber Notes

A collection of study notes and resources from my journey into **cybersecurity** and **networking**.

## About
This repository documents my learning process in IT security and networking.  
It includes summaries, useful commands, and practical tips based on **TryHackMe** labs, online resources, and personal practice.

## Topics Covered
- **Networking Fundamentals** – OSI model, IP addressing, protocols
- **Linux Essentials** – Basic commands, file permissions, system navigation
- **Cybersecurity Basics** – Hands-on exercises, attack/defense concepts

## Goal
To openly share my progress while building a solid foundation in IT security.

## File Structure
- `network_basics.md` – Notes on networking fundamentals
- `linux_commands.md` – Useful Linux commands and scripts
- `tryhackme_notes.md` – Writeups and notes from TryHackMe exercises
